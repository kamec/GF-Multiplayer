To get generated code perform folowing actions:

  1. Choose needed multiplier algorithm.
    • "simple" is for right-to-left shift-and-add.
    • "reyhani-masoleh" is for Reyhani-Masoleh algorithm that performs multiplication using matrices.
    • "karatsuba" is for binary implementation of Karatsuba-Ofman multiplication.
  2. Choose language of algorithm realization.
  3. Provide a monic irreducible polynomial for Galoix field.
