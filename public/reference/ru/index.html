Для того чтобы получить результат, выполните следующие действия:

  1. Выберите алгоритм умножения, реализацию которого вы хотете получить.
    • "simple" - алгоритм right-to-left shift-and-add.
    • "reyhani-masoleh" - алгоритм Рейхани-Мазолеха, использующий матрицы для быстрого умножения.
    • "karatsuba" - алгоритм двоичного умножения Карацубы-Офмана.
  2. Выберите язык реализации алгоритма.
  3. Укажите образующий полином поля Галуа, в котором будет производиться умножение.
